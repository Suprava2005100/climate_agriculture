## 🔄 Preprocessing Workflow

### In `preprocess.ipynb` (step-by-step):
1. **Load data** from `raw/`
2. **Inspect** dataset:
   - Column names & datatypes
   - Missing values
   - Summary statistics
3. **Clean data**
   - Remove duplicates
   - Handle missing values:
     - Numeric → filled with mean
     - Categorical → filled with `"Unknown"`
4. **Standardize column names**, if needed
5. **Save cleaned dataset** to `processed/`
6. **Verify** that the processed file exists

### In `preprocess.py` (script version):
Same process is automated for repeatable execution.
